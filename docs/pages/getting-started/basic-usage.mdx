# Basic Usage

To get started using the [@ibnlanre/builder][1] package, it is essential to understand the basic concepts and usage patterns involved in working with the library. This guide will walk you through the fundamental steps required to create a [builder][3] object, access values, and retrieve keys using the package.

The [@ibnlanre/builder][1] package provides a simple and efficient way to generate and manage keys within your application. It requires a [register][4] to be passed as an argument to the [createBuilder][2] function, which can be imported from the package. In turn, it generates a builder which serves as a bridge between the [register][4] object and the user, providing a convenient way to interact with the data structure.

The following steps outline how to quickly get started using the [@ibnlanre/builder][1] package:

1. Import the [createBuilder][2] function from the library:

    ```typescript
    import { createBuilder } from '@ibnlanre/builder';
    ```

2. Create a [register][4] object with nested keys and values:

    ```typescript
    const register = {
      foo: {
        baz: (id: number) => `/bazaar/${id}`,
        bar: 10,
      },
    }
    ```

3. Create a [builder][3] object using the [createBuilder][2] function:

    ```typescript
    const builder = createBuilder(register, ["root", "node"]);
    ```

4. Access the [register][4] values using the [builder][3] object:

    ```typescript
    // Accessing the register values
    builder.$use; // { foo: { baz: [Function: baz], bar: 10 } }

    // Accessing a nested function value
    builder.$use.foo.baz(12); // "/bazaar/12"

    // Accessing a nested primitive value
    builder.$use.foo.bar; // 10
    ```

5. Generate keys using the [builder][3] object:

    ```typescript
    // Retrieving the root key
    builder.$get(); // ["root", "node"]

    // Accessing a nested function key
    builder.foo.baz.$use(12); // ["root", "node", "foo", "baz", 12]

    // Accessing a nested primitive key
    builder.foo.bar.$use(); // ["root", "node", "foo", "bar"]
    ```
